# Задание 02 (Процессы)

## Условие

Написать программу, похожую на командный интерпретатор.

При запуске выводится приглашение, и пользователь вводит имя и аргументы программы, которую желает запустить. Если программа не найдена, выводится соответствующее сообщение.

После завершения запущенной программы, ваш командный интерпретатор снова выводит приглашение.

Командный интерпретатор должен запускать как программы, лежащие в том же каталоге, так и системные программы. Напишите 2-3 простых программы (например, программа для вычисления суммы аргументов, «склеивания» строк, поиска наибольшего значения или наибольшей длины строки и т.д.) для тестирования.

## Сборка

### Решение (интерпретатор)

```shell
make solution
```

### Тестовые программы

```shell
make examples
```

## Запуск

### Решение (интерпретатор)

- Неинтерактивный режим
`./shell FILE [OPTIONS...]`
- Интерактивный режим
`./shell`

```shell
./shell ls ~
./shell
```

> Для выхода из интерпертатора необходимо ввести `exit` или воспользоваться `^D`

> Для получения возвращаемого значения последней запущенной программы необходимо ввести `?`

> Все аргументы внутри интерпретатора разделяются при помощи пробелов, для ввода аргумента с пробелом необходимо либо экранировать пробел (`\ `), либо использовать двойные кавычки (`""`)

> Интерпретатор поддерживает следующие escape-последовательности: `\n`, `\t`, `\\`, `\'`, `\"`, `\v`, `\r`, `\0`, `\ `

### Тестовые программы

#### sum

`./sum [NUM...]`

```shell
./sum 123 444
```

#### print-args

`./print-args [ARGS...]`

```shell
./print-args abc 123
```
