# Задание 04 (Каналы)

## Условие

Усовершенствовать командный интерпретатор так, чтобы можно было использовать перенаправление ввода и вывода: перенаправить вывод одного процесса на ввод другого и т.д. Количество процессов в конвейере не ограничивать. Также предусмотреть возможность чтения данных из файла и вывода в файл.

## Сборка

### Решение (интерпретатор)

```shell
make solution
```

### Тестовые программы

```shell
make examples
```

## Запуск

### Решение (интерпретатор)

- Неинтерактивный режим
`./shell FILE [OPTIONS...]`
- Интерактивный режим
`./shell`

```shell
./shell ls ~
./shell
```

> Для выхода из интерпертатора необходимо ввести `exit` или воспользоваться `^D`

> Для получения возвращаемого значения последней запущенной программы необходимо ввести `?`

> Все аргументы внутри интерпретатора разделяются при помощи пробелов, для ввода аргумента с пробелом необходимо либо экранировать пробел (`\ `), либо использовать двойные кавычки (`""`)

> Интерпретатор поддерживает следующие escape-последовательности: `\n`, `\t`, `\\`, `\'`, `\"`, `\v`, `\r`, `\0`, `\ `

> Интерпретатор поддерживает конвейер команд (`|`), например `cat 1.txt | head -n 3 | tail -n 1`

> Интерпретатор поддерживает перенаправление ввода и вывода (`>`, `>>`, `<`), например `head -n 3 < 1.txt | tail -n 1 > 2.txt`

### Тестовые программы

#### sum

`./sum [NUM...]`

```shell
./sum 123 444
```

#### print-args

`./print-args [ARGS...]`

```shell
./print-args abc 123
```
