# Задание 12 (UDP сокеты)

## Условие

Реализовать чат для двух клиентов с использованием UDP-сокетов. Предусмотреть "асинхронное" взаимодействие (отправку сообщений в любом порядке)

## Сборка

```shell
make client
```

## Запуск

> `LOCAL_PORT` - Порт, на котором запустится клиент

> `PEER_PORT` - Порт, который будет слушать клиент (и отправлять сообщения)

> `PEER_ADDR` - Адрес второго клиента (по умолчанию `127.0.0.1`)

`./client LOCAL_PORT PEER_PORT [-a|--address PEER_ADDR]`

```shell
./client 51112 51111
./client 51111 51112 -a 192.168.2.169
```
